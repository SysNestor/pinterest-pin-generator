Pinterest Pin Generator

This is a personal/internal tool used to generate Pinterest pin metadata (titles, descriptions, hashtags, image hooks, and board suggestions) and optionally publish pins to the connected user’s Pinterest account after OAuth authorization.

Key features

  • Generate Pinterest-native pin ideas optimized for search and clicks.
  • After connecting Pinterest via OAuth, list boards and publish pins with user-uploaded images.
  • User can manually add a destination link per pin.
